BenzConfig PWA
BenzConfig — это Progressive Web App (PWA) для расчёта расхода топлива в летнем и зимнем режимах. 
Приложение адаптировано для iOS, с минималистичным дизайном, поддержкой настройки пропорций и норм расхода.

Возможности
Расчёт расхода топлива для Летнего и Зимнего режимов.
Отображение детализации расхода по городу и трассе.
Настройка пропорций город/трасса и норм расхода.
Кнопка «О программе» с информацией об авторе, лицензии и исходном коде.
PWA-версия для iOS с возможностью добавления на домашний экран.
Модальное окно с аккуратным дизайном и поддержкой haptic feedback.

Технологии
HTML, CSS, JavaScript
PWA (manifest.json + service worker)
Совместимость с iOS и Android через браузер

Установка / Запуск
Для PWA на iOS:
Откройте сайт в Safari.
Нажмите «Поделиться» → «На экран Домой».

Использование
Введите пробег в поле «Летний» или «Зимний».
Настройте пропорции и нормы расхода при необходимости.
Нажмите «Рассчитать», чтобы увидеть полный расход с детализацией.
Для информации об приложении нажмите «О программе».
