BenzConfig PWA <br>
BenzConfig — это Progressive Web App (PWA) для расчёта расхода топлива в летнем и зимнем режимах.  <br>
Приложение адаптировано для iOS, с минималистичным дизайном, поддержкой настройки пропорций и норм расхода. <br> <br>

Возможности <br>
Расчёт расхода топлива для Летнего и Зимнего режимов. <br>
Отображение детализации расхода по городу и трассе. <br>
Настройка пропорций город/трасса и норм расхода. <br>
Кнопка «О программе» с информацией об авторе, лицензии и исходном коде. <br>
PWA-версия для iOS с возможностью добавления на домашний экран. <br>
Модальное окно с аккуратным дизайном и поддержкой haptic feedback. <br> <br>

Технологии <br>
HTML, CSS, JavaScript <br>
PWA (manifest.json + service worker) <br>
Совместимость с iOS и Android через браузер <br> <br>

Установка / Запуск <br>
Для PWA на iOS: <br>
Откройте сайт в Safari. <br>
Нажмите «Поделиться» → «На экран Домой». <br> <br>

Использование <br>
Введите пробег в поле «Летний» или «Зимний». <br>
Настройте пропорции и нормы расхода при необходимости. <br>
Нажмите «Рассчитать», чтобы увидеть полный расход с детализацией. <br>
Для информации об приложении нажмите «О программе». <br> <br>
