# BenzConfig PWA 🚗⛽

Прогрессивное веб-приложение для расчёта расхода топлива **летом и зимой** с детальной разбивкой по городу и трассе.  

![Главный экран BenzConfig](screenshot-main.png)

---

## ⚡ Основные возможности

- Расчёт расхода топлива для **летнего** и **зимнего** периодов  
- Детализация:
  - Пробег по городу / трассе  
  - Нормы расхода  
  - Пропорции город/трасса  
- Настройка пользовательских норм и пропорций  
- Кнопка **О программе** с информацией об авторе и лицензии  
- Haptic feedback на iOS  
- Блокировка ориентации экрана  
- Установка на экран как PWA (iPhone/iPad)

## 📲 Установка на iOS

1. Откройте PWA в Safari на iPhone/iPad  
2. Нажмите **Поделиться → На экран «Домой»**  
3. Приложение появится на домашнем экране
