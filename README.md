### BenzConfig PWA
Простой и удобный калькулятор расхода топлива <br> 
для **летнего** и **зимнего** режимов, <br> 
с детальной разбивкой по городу и трассе.  
Попробовать [BCPWA](https://benzenergy.github.io/BenzConfig-PWA)

---

### Особенности
Расчёт расхода топлива с детализацией  
Настройка норм и пропорций город/трасса  
Кнопка **О программе** с лицензией и автором  
Поддержка PWA на iOS (установка на экран «Домой»)  
Haptic feedback и блокировка ориентации экрана
